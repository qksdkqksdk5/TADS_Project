AWS VPC + EC2 배포 가이드 (2a / 2c)

1. VPC 생성

CIDR: 10.3.0.0/16

2. Subnet 구성

Public Subnet (2a)

  • CIDR: 10.3.1.0/24
  • Public IP 자동 할당 활성화

Private Subnet (2c)

  • CIDR: 10.3.2.0/24

3. Internet Gateway

VPC에 연결 (Attach)

4. Route Table

Public

0.0.0.0/0 → IGW

Private

0.0.0.0/0 → NAT Gateway

5. NAT Gateway

  • Public Subnet에 생성
  • Elastic IP 필수

6. EC2 구성

Public EC2

포트용도

80HTTP
443HTTPS
5000Flask

Private EC2

포트허용 대상

7. EC2 접속

핵심 요약

  • Public: 외부 접근
  • Private: DB 보호
  • NAT: Private 인터넷 연결
  • IGW: Public 인터넷 연결
